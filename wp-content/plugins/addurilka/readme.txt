Plugin for verify a link indexing and adding it to the search engines.

== Description ==

Простой плагин, который добавляет на панель администратора виджет для проверки и добавления публикаций в поисковых системах.<br>
Можно посмотреть проиндексирована ли ссылка текущей страницы в ...<br>
...и добавить её в случае отсутствия в поисковые системы.

A simple plugin that adds a widget to the admin panel for add and verify the site links to search engine.

== Installation ==

Процесс инсталляции плагина стандартен для WordPress.<br>
Установите плагин в Wordpress используя установщик плагинов.<br>
Активируйте плагин в вкладке Плагины.<br>
Перейдите в настройки плагина и настройте его.

Download plugin through the wordpress included plugin downloader.<br>
Activate plugin in plugin tab.<br>
Go in the plugin settings and configure it.

== Frequently Asked Questions ==

= Какие планы на развитие? =

* Добавлять другие поисковые системы
* Добавить настройку уровня доступа к виджету

Нужна помощь от программистов для доработки автоматической проверки ссылок (работает нестабильно).
